# Desafio-Front-End-II
Aplicação web desenvolvida com HTML, CSS e JavaScript, que permite o preenchimento, validação e armazenamento local de dados de um formulário. O sistema gera automaticamente um arquivo data.json com as informações inseridas, demonstrando o uso de validação de campos, LocalStorage e manipulação de arquivos no navegador.

Funcionalidades

Preenchimento de formulário com campos: Nome, Sobrenome, Email e Idade.

Validação em tempo real de cada campo:

Nome e Sobrenome: 3 a 50 caracteres.

Email: formato válido.

Idade: número inteiro positivo menor que 120.

Exibição dos dados preenchidos na página de confirmação.

Geração automática de arquivo data.json para download.

Armazenamento do último registro no LocalStorage para visualização na página inicial.

Carregamento opcional de um arquivo data.json existente.

Tecnologias Utilizadas

HTML5 – estrutura das páginas.

CSS3 – estilização moderna e responsiva.

JavaScript (ES6) – validação de formulário, manipulação de LocalStorage e download de arquivos.
