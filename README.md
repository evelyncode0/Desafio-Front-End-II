# Desafio-Front-End-II
Aplicação web desenvolvida com HTML, CSS e JavaScript, que permite o preenchimento, validação e armazenamento local de dados de um formulário. O sistema gera automaticamente um arquivo data.json com as informações inseridas, demonstrando o uso de validação de campos, LocalStorage e manipulação de arquivos no navegador.

⚙️ • #Funcionalidades

Formulário com campos obrigatórios: Nome, Sobrenome, Email e Idade.
Validação em tempo real:
Nome e Sobrenome: 3 a 50 caracteres.
Email: formato válido.
Idade: número inteiro positivo menor que 120.
Página de confirmação com revisão dos dados.
Geração automática de data.json para download.
Último registro salvo é exibido na página inicial.
Opção de carregar um arquivo data.json existente.
